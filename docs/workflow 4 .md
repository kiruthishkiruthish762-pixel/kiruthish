#Step 1

Open your browser and go to your n8n instance.

Sign in with your credentials or create a new account if needed.

Once logged in, click Create Workflow to start the new workflow.

#step 2
n the n8n editor, click the + button to add a new node.

Search for When Clicking ‘Execute Workflow’ and select it.

Configure this node to listen for a button click that will trigger the workflow.

#step 3
Click on the + button again to add another node.

Search for Send a Message and select Gmail.

Configure the Send a Message node to send an email when the trigger event occurs.
